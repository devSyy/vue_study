# vue-study

## Project setup
```
vue create vue-study
npm install
npm i --save vue-router
npm install --save axios
npm install vuex --save
```
### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
